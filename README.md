# mgp-blog-images

Category images for the **Margot Gibson Photography** blog — one per content category.
The blog pipeline (`mgp-blog`) embeds these by raw URL at import. To use Margot's real
photos, **replace any file with the same filename** and the blog picks it up automatically.
